About

This project aims to demonstrate benefits of transfer learning. For observation of the benefits, we used two classifiers for sign language. Transfer learning is achieved with the help of pretrained model for Fassion MNIST dataset

Dataset

  • Dataset consists of 1400 images equally distributed among following 7 classes
    • Fantastic
    • Hi five
    • Strength
    • Thubms down
    • Thumbs up
    • Victory
    • Yolo
  • All images are in '.jpg' format and have dimensions (2448 x 2448)

Technologies

  • For model creation
    • Python
    • Matplotlib
    • Keras
  • For model serving
    • Django
    • Bootstrap
    • Scikit-image

Observations

  • Model obtained by transfer learning is more accurate for testing new samples
  • Training time is much less for transfer learning because of almost half the number of trainable parameters
  • Transfer learning achieved a higher accuracy is less epochs
